### 📖 Project Overview
**PhishGuard** is a premium cybersecurity education and detection tool developed by the **ISTE Co-ordinators** of **Mohan Babu University (MBU)**. This repository houses the source code for a comprehensive phishing-awareness web application that helps users identify malicious URLs and learn about web safety through an interactive, modern interface.

### ✨ Key Features
- **🔍 Intelligent URL Analyzer**: Rule-based detection engine checking for typosquatting, suspicious TLDs, and SSL vulnerabilities.
- **🛡️ Real-time Protection**: Instant safety feedback (Safe, Warning, Danger) with detailed risk explanations.
- **📧 Security Newsletter**: Integrated subscription system for the latest cybersecurity updates.
- **🎨 Premium UI/UX**: A state-of-the-art glassmorphic design optimized for professional impact.
- **⚡ Serverless Backend**: Robust API handled via Netlify Functions for seamless scalability.

### 🚀 Getting Started

#### Prerequisites
- Node.js 18.x or higher
- npm (Node Package Manager)

#### Local Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/Team-Nova.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start the local server**:
   ```bash
   npm start
   ```
4. **Access the application**:
   Open `http://localhost:3000` in your browser.
